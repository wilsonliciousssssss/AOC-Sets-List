# Add a gig — the 30-second version

1. **Make the sheet.** Copy `sets/_TEMPLATE.html` (or generate one with your
   dj-set-preparation skill). Fill in:
   - the **`AO-GIG`** comment at the top → card info (title, venue, date, channel…)
   - the **`SET`** object in the script → the tracklist.
2. **Name it** `set-6.html` (or a venue/date name).
3. **Upload it into the `sets/` folder** on GitHub → *Add file → Upload files*.
4. **Done.** Refresh the homepage — the card is there. Past dates auto-move to
   "Past Sets".

Card colour = the `channel` field: `lime · cobalt · orange · magenta · teal · violet`.

No manifest, no build, no code to touch.
